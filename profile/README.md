## 모각테(MogakTae)
모각테는 “**모여서 각자 코딩 테스트 준비**”의 줄임말로,<br>
코딩 테스트를 준비하는 사람들이 **함께 목표를 설정하고 도전할 수 있도록 돕는 챌린지 기반의 서비스**입니다.

<br>

## 팀원 소개
- PM : **한준서** 👑
- FrontEnd : 조하은, 최민지
- BackEnd : 한준서
